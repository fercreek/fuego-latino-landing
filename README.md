# Fuego Latino Dance Studio Landing

Landing page en Next.js + Tailwind para clases de salsa, bachata, urbano y heels.

## Stack
- Next.js (App Router, TS)
- Tailwind CSS v4
- framer-motion para animaciones
- react-scroll-parallax para efectos suaves

## Scripts
- `npm run dev` arranca en http://localhost:3000
- `npm run lint` revisa ESLint
- `npm run build` compila para producción
